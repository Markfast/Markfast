Your Guide to Markdown

Markdown is a markup language designed to be legible and easy to convert to HTML. The Markfast editor is currently using the CommonMark Spec v0.28. The following is a guide to Markdown's syntax.

Inline HTML

Inline HTML tags are parsed as HTML, meaning they can be used to structure a Markdown document. For instance, to format some text as inline code, you could use Markdown's native method of placing a backtick at each end of the inline code, or you could wrap the inline code in <code> tags.

Markdown Presentation Hotkeys 
`Inline code`
 Inline code 
<code>Inline code</code>
 Inline code

Headings

To use HTML h1-h6 headings, use pound signs at the start of the line. For instance, use # for h1, ## for h2, and so forth up to ###### for h6.

Markdown Presentation Hotkeys 
# Heading 1

Heading 1

 1 
## Heading 2

Heading 2

 
### Heading 3

Heading 3

 
#### Heading 4

Heading 4

 
##### Heading 5
Heading 5

Emphasis

Markdown supports bolding, italicizing, and strikethrough. Any of these can be used in tandem with each other.

Markdown Presentation Hotkeys 
**This is bold.**
__This is also bold.__
 This is bold.
This is also bold. 		B 
*This is italicized.*
_This is also italicized._
 This is italicized.
This is also italicized. 		I 
~~This is struck through.~~
 This is struck through.
